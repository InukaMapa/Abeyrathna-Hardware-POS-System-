# Abeyrathna-Hardware-POS-System-
This project is a Smart Point of Sale (POS) System developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js). It is designed to help businesses efficiently manage products, inventory, billing, and sales operations with automation and smart features.

The system includes inventory tracking, billing management, profit analysis, and AI-powered enhancements like OCR for supplier bill processing.

✨ Features
🛒 Product & Inventory Management
Add, update, and manage products/items
Real-time inventory tracking
Manual & automatic stock updates via supplier bills
Identify out-of-stock and unavailable items
💰 Sales & Billing
Generate customer bills/invoices
Barcode generation and download
Apply discounts and promotional offers
Cashier session management (login/logout)
📊 Reports & Analytics
Weekly profit calculation and reports
Most-selling items analysis
Administrative reporting dashboard
🔔 Notifications & Verification
Automated bill payment reminders (7 days before deadline)
Payment slip verification system
🤖 Smart Features
OCR + ML integration for supplier bill processing
Cloud-based scalable architecture
🛠️ Tech Stack

Frontend:
-React.js
-Tailwind CSS / Bootstrap (optional)

Backend:
-Node.js
-Express.js

Database:
-MongoDB (with optional Supabase/PostgreSQL support)

Cloud & Services:
